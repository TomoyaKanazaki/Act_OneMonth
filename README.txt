◆作品説明
・居合斬りを駆使して敵を斬る！

◆現在の実装内容
・コントローラ未接続状態のエラー表示
・簡易的な操作説明
・タイトルの演出
・ゲームクリアとゲームオーバー
・居合斬りの構え
・ジャンプと重力による下降
・居合ダッシュの軌跡上の敵を倒すことができる
・タイトル→ゲーム→リザルト→タイトルのゲームループ

◆操作方法◆
[Xboxコントローラ]
左スティック	: 移動
右スティック	: 傾けた方向に攻撃
LBボタン		: ジャンプ
RBボタン		: (押しながら)攻撃がダッシュに変化
Aボタン			: 画面遷移
