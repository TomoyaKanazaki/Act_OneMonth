◆作品説明
・直感的で気持ちのいいアクションを追求！

◆工夫した点
・プレイヤーの攻撃の当たり判定に線分の当たり判定を実装
・初めての人型モデルで初めてのモーションを作成
・明るく派手なエフェクトでも何が起きているのかわかりやすいように調整をしました。

◆反省点
・初めての試みに時間をかけすぎてしまった。スケジューリングを大切にする。
・度重なる仕様変更を行った。作り始める前の段階でもっとしっかりと仕様を固める。


◆今後の展望
・ボスの行動パターンを増やす
・敵の種類を増やし遊びに幅を持たせる
・「気持ちのいい演出」をさらに追及

◆操作方法◆
[Xboxコントローラ]

右スティック	: 傾けた方向に攻撃
LBボタン		: ジャンプ
Aボタン			: 画面遷移
