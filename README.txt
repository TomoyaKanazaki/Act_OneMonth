◆作品説明
・居合斬りを駆使して敵を斬る！

◆現在の実装内容
・コントローラ未接続状態のエラー表示
・簡易的な操作説明
・タイトルの演出
・ゲームクリアとゲームオーバー
・定位置を徘徊する敵
・画面内にいる間プレイヤーに向かって動く敵
・集中状態中に敵が減速する処理
・居合斬りの構え
・ジャンプと重力による下降
・居合ダッシュの軌跡上の敵を倒すことができる
・「ぼす」と書いてある敵を倒すとリザルトに遷移します。
・「ぼす」の周りに飛んでる「こぶん」を倒さないと「ぼす」を倒せません
・タイトル→ゲーム→リザルト→タイトルのゲームループ

◆12/13のカレッジから変更した点
・チュートリアル中に右に進めないようにしました。
・フォグを追加
・集中状態で敵にエフェクトを付けました。
・集中状態中にのみフォグで世界を暗くしました。
・移動先の軌跡を表示しました。
・カーソル操作での敵の撃破を実装

◆操作方法◆
[Xboxコントローラ]
・左スティック	: プレイヤーの移動(横軸のみ)
・右スティック	: (集中状態中)ダッシュ方向の設定
・Rボタン		: (集中状態中)居合ダッシュ
・LTボタン		: 集中状態切り替え
・Lボタン		: ジャンプ
・Aボタン		: タイトル、リザルト画面から次の画面への遷移
